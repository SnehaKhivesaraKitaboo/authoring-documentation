Author Top Bar

Shared Rich Text Editor (RTE) toolbar that appears when authors select or focus text in contenteditable areas. Used by Elements, Layout content, and Widgets for formatting (font, size, bold, italic, alignment, lists, links, image, equation, special characters).

Overview Architecture Features Flow Integration Files Behaviors

Overview

The Author Top Bar is implemented as the Docked Text Editor Panel: an AngularJS element <dockedtexteditorpanel id="editor" class="docked-text-editor"> that loads the template templates/textEditor.html. It provides a horizontal bar of controls for font family, font size, bold, italic, underline, text/background color, alignment, line height, lists, indent, hyperlinks (internal/external/glossary), insert image, math equation, split text, and special character insertion. Formatting is applied via document.execCommand and shared logic in editor/floatingTextPanel.js.

Integration with Elements, Layout, TOC, Widgets

  • Elements — Title, Header, Text (paragraph), Callout Box, Thumbs Card Header, Lesson Opener use contenteditable; selecting text shows the top bar and formatting applies to their fieldData (e.g. header, body, title).
  • Layout — Layout column content that is contenteditable uses the same RTE; the panel appears when focus is inside that content.
  • TOC — TOC does not use the top bar; TOC entries are plain text synced from canvas headings.
  • Widgets — Extended Response (ITC), Table GO, Graphic Organizer, Annotation, and other widgets with rich-text fields show the top bar when the author selects text. Some buttons are disabled in certain contexts (e.g. Math in ITC, alignment in Title) via classes like cursorNotAllowedInPanel, disableOnlyTable.

Known Behaviors & Notes

  • Single global panel instance; active contenteditable is determined by document.getSelection() and parent data-type.
  • Panel can be hidden with $(".docked-text-editor").hide() (e.g. preview mode).
  • Open dropdowns (".open") are closed when clicking outside or applying format.
  • Font family and font size selects are reset on certain selection changes (#font-family → "NA", #font-size → " ").
  • Title element can disable alignment/justify via cursorNotAllowedInPane.
  • Apply All: applying format may set ignoreApplyAll = true and strip inline formatting so the element follows global style.
  • Italic uses custom span (.italics/.nonItalics) in some code paths.
  • After format, refreshMathFieldsForStyleChange('#target') and ng-model trigger keep saved JSON in sync.
