Audio Element

An HTML5 audio player component supporting audio file upload, optional caption, outline styles, background playback mode, and custom skin selection.

Overview Architecture Data Model UML Class Sequence Component Flow

PlayerNative HTML5 <audio>

Key Capabilities

  • Audio file upload (MP3, WAV, OGG, etc.)
  • Native HTML5 browser audio player
  • Custom skin selection via getSkinName()
  • Background (ambient) playback mode
  • Optional figcaption text
  • Outline / border style toggle
  • Background color for outline box
  • Audio alignment
Architecture Note: Like Image and Video, the Audio directive lives in editor/audio-directive.js for reuse across multiple contexts.

Settings Panel Options

Audio Options

  • Upload audio file / Replace
  • Audio alignment
  • Skin selection

Display Options

  • Show/hide caption
  • Outline style toggle
  • Outline background color
  • Background playback mode toggle
Background Playback: When playinBackground is enabled (playInBackground attribute on the element), the audio plays as ambient background music in the reader, without showing visible player controls. This is useful for chapter atmosphere audio.
