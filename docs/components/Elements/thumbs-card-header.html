Thumbs Card Header Element

A card-style section header with a selectable icon (48×48px) and an editable heading, supporting custom background and stroke colors. Designed to visually introduce categorized content sections.

Key Capabilities

  • Selectable icon from predefined icon library
  • Editable heading text (contenteditable)
  • Custom header background color
  • Custom header stroke/border color
  • Fixed 48×48px icon display
  • Compact, horizontal card layout
Purpose: The Thumbs Card Header provides a visually distinctive section header that combines an icon with a title. It is typically used to introduce categorized sections (e.g., "Did You Know?", "Quick Tip", "Key Concept") where the icon conveys the content type at a glance.

Icon Array Structure

settings.thumbsCardIcons = [
  { icon: "images/thumbs-card-icon-1.svg" },
  { icon: "images/thumbs-card-icon-2.svg" },
  { icon: "images/thumbs-card-icon-3.svg" },
  // ... more icons
];
settings.thumbsCardActiveIcons = 0; // Index of active icon
Icon Selection: The active icon is selected by index (thumbsCardActiveIcons) from the thumbsCardIcons array. The settings panel shows thumbnail previews of all available icons; clicking one updates the active index.
