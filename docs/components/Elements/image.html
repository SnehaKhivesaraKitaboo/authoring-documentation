Image Element

A standalone media image component supporting upload, alignment, optional header text, caption, outline styles, and accessibility alt text.

Overview Architecture Data Model

Key Capabilities

  • Image upload via file picker or media library
  • Image alignment: left, center, right
  • Optional header text above image
  • Optional caption text below image
  • Outline / border style toggle
  • Background color for outline box
  • Accessibility alt text field
  • Style variants (style1, style2)
Architecture Note: Unlike most elements, the Image element's AngularJS directive is defined in editor/image-directive.js (not inside the templates/ folder). This is because the Image directive is reused in multiple contexts — standalone Image element, within layout columns, and as an inline image inside paragraphs.
