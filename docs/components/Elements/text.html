Text (Paragraph) Element

The primary rich-text content component supporting body paragraphs, blockquotes, pull-quotes, and label types, with inline image support and a floating text editor toolbar.

Key Capabilities

  • Full rich-text editing via floating text toolbar
  • Paragraph, blockquote, pull-quote variants
  • Inline image insertion & split-paragraph layout
  • Paragraph number display toggle
  • Label type (Primary/Secondary question label)
  • Author name field for blockquote attribution
  • Apply All / Ignore Apply All
  • Math equation support
Purpose: The Text element is the most commonly used element — it provides the main body content area for page text. It supports rich formatting (bold, italic, lists, tables, links, images) via the floating text editor toolbar, and multiple layout styles including blockquotes with author attribution.

Floating Text Toolbar

The Text element activates the shared floating text editor toolbar (floatingTextPanel.js) when the author selects text inside the content area. The toolbar provides:

  • Bold, Italic, Underline, Strikethrough
  • Font family and size selectors
  • Text color and background color pickers
  • Text alignment (left, center, right, justify)
  • Ordered and unordered lists
  • Hyperlink insertion
  • Superscript / Subscript
  • Math equation insertion
  • Inline image upload

Content Storage: The fieldData.header field stores raw HTML (produced by contenteditable). This includes inline styles, formatting tags, and embedded image references. The save/load system must handle HTML entity encoding carefully.
