Lesson Opener Components

A multi-style lesson introduction component with 4 predefined prompt types — Learning Prompt, Make a Connection, My Learning Goals, and Key Terms — each with a unique icon, label, color scheme, and editable body text.

Key Capabilities

  • 4 predefined lesson prompt types
  • Each type has a unique icon and label
  • Editable body text (rich HTML)
  • Custom background color per style
  • Custom border/stroke color per style
  • Outline style support
  • Flexible placement (left, right, full-width)
Purpose: The Lesson Opener element provides structured, pedagogically-labeled prompts at the beginning of lessons. Each style represents a specific type of learning activity: activating prior knowledge, making connections, goal setting, or vocabulary introduction. The fixed labels and icons ensure consistent visual language across all content.

Fixed Labels: The text labels ("Learning Prompt", "Make a Connection", etc.) are hard-coded in the HTML template — they are not editable by the author. Only the body text (fieldData.paragraph) is editable. This ensures pedagogical consistency across all books.
