Table Element

A fully editable HTML table component with resizable columns, row/column insert/delete, cell merging, theme styles, header text, caption, and label type support.

Key Capabilities

  • Dynamic row/column creation dialog
  • Insert row above/below, insert column left/right
  • Delete rows and columns
  • Cell merge (Merge All) and Unmerge
  • Resizable columns (drag handle)
  • Multiple visual themes
  • Optional header text above table
  • Optional figcaption below table
  • Label type (Primary/Secondary question label)
  • Outline / background color styling
  • Math equation support in cells

Table Operations

Row Operations

Right-click context menu (.vmenu) on any row:

  • Insert Above — globalScope.appendRow(rowIndex - 1, 'up')
  • Insert Below — globalScope.appendRow(rowIndex, 'down')
  • Delete — globalScope.deleteRows(rowIndex)

Column Operations

Right-click context menu on any column:

  • Insert Left — globalScope.appendColumn(colIndex - 1, 'left')
  • Insert Right — globalScope.appendColumn(colIndex - 1, 'right')
  • Delete — globalScope.deleteColumns(colIndex - 1)

Cell Merge

  • Merge All — Merges selected cells into one (mergeAll())
  • Unmerge — Splits merged cell back (unmergeAll())

Column Resize

  • Drag column border to resize
  • Powered by resizable.js
  • Column widths persisted in tableData

Table Creation Dialog

When a new Table element is first dropped, a dialog appears to define the initial dimensions:

Rows: ng-model="fieldData.noOfRows" (1–20, step: 1)
Cols: ng-model="fieldData.noOfCol"  (1–20, step: 1)
isNoOfRowColChangedFlag = true → triggers table rebuild
