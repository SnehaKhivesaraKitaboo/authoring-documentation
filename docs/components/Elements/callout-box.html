Callout Box Element

A highlighted content callout component (internally named Sidebar Component) that presents supplementary information with a header, body text, optional image, and multiple visual styles including auto-color and icon variants.

Key Capabilities

  • 4 visual style variants (style1–style4)
  • Editable header / title text
  • Editable body text paragraph
  • Optional embedded image with caption
  • Auto-color mode for background
  • Custom header background color
  • Custom body background color
  • Outline / stroke color
  • Style3 icon (pill/note icon)
  • Style4 icon picker (multiple icon options)
  • Left/Right/Full placement
Purpose: The Callout Box highlights supplementary content, tips, notes, warnings, or sidebar information alongside main page content. It is flexible enough to act as a simple text callout (style1/style2) or an icon-annotated note block (style3/style4).

Style Variants

style1 — Header + Body + Image

Standard callout with colored header bar, body text area, and optional image. Header background color is configurable. Border applied as 1px solid headerBgColor.

style2 — Body + Image Only

Callout without a separate header section — just body text and optional image. No top-bar color.

style3 — Pill Icon Callout

Compact callout with a fixed circular pill/note icon (Sidebar-Style-3-icon.png) on the left and body text on the right. No header. Uses sidebarBgColor and sidebarStrokeColor for theming.

style4 — Icon Picker Callout

Callout with a selectable icon (from style4Icons array), a label text derived from the icon definition, and body text. The icon and label are predefined; the author selects from available options.

Auto Color Mode: When allowAutoColor is enabled, the component uses a predefined green-tinted color scheme (rgb(244, 248, 233)) for the background, overriding manual color settings.
