Situation Header

Situation Header Element

A visually distinct context-setting prompt box with a decorative arrow icon and a rich-text editable area, designed to present scenario introductions and situational prompts at the start of a learning section.

Key Capabilities

  • Decorative arrow icon (non-interactive)
  • Rich-text editable body area
  • Floating text toolbar for formatting
  • Math equation support
  • Alt text for accessibility
  • Preview mode rendering
  • Fixed purple-tinted visual design
Purpose: The Situation Header is a specialized element used to introduce a learning scenario, real-world context, or situational problem before related questions or activities. Its distinctive rounded purple box with an arrow icon visually signals "here is the situation" to students.

Simple Data Model: The Situation Header has one of the simplest data models — all editable content is in fieldData.textArea as rich HTML. The visual design (background color, border radius, arrow icon) is entirely CSS/HTML defined and not customizable by the author.

Visual Design

situation-box.html — Fixed styling
<div class="situation-box-container"
  style="background-color: #f3ecfe; border-radius: 16px;">
  <div class="situation-box-title-container"
    style="display: flex; padding: 16px 16px 0px 16px">
    <!-- Arrow icon (non-draggable, pointer-events: none) -->
    <img src="images/arrow.svg" alt="..." draggable="false"/>
    <!-- Rich text area -->
    <div contenteditable="true"
      ng-model="fieldData.textArea"
      style="font-family: Roboto; font-size: 16px; line-height: 24px;
             min-height: 72px;">
    </div>
  </div>
</div>
Non-Customizable Design: The background color (#f3ecfe), border radius (16px), padding, arrow icon, font family, font size, and line height are all hard-coded in the template HTML using inline styles. Authors cannot change the visual theme of this element.
