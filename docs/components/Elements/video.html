Video Element

A video media component supporting MP4 upload, YouTube/Kaltura embeds, transcript display, closed captions (CC), and optional header/caption text.

Overview Architecture Data Model Video Sources

Overview

Component Identity

dataTypevideo

PlayerVideo.js (video-js)

Key Capabilities

  • MP4 video upload (direct file upload)
  • YouTube embed via iframe
  • Kaltura embed via iframe
  • Transcript panel toggle
  • Closed Captions (CC) toggle
  • Multiple language caption tracks
  • Optional header text
  • Optional figcaption
  • Outline / border style
  • In-player settings overlay (transcript & CC)
Architecture Note: Like the Image element, the Video directive lives in editor/video-directive.js so it can be reused across standalone use and within layout columns.

Video Sources

MP4 Upload

Direct MP4 file upload — renders as a Video.js player with controls. Supports:

  • Playback controls (play/pause/volume/fullscreen)
  • Closed caption track injection (VTT files)
  • Transcript panel below video
  • Multi-language caption support

External Embeds

YouTube and Kaltura videos render as responsive iframes:

  • 16:9 responsive container (embed-responsive-16by9)
  • URL sanitized via | trusted filter
  • CC handled natively by platform

In-Player Settings Overlay

An overlay panel inside the video element allows authors to toggle Transcript and CC directly on the canvas:

In-player overlay controls
Transcript toggle  → fieldData.settings.showTranscript (boolean)
CC toggle          → fieldData.settings.showClosedCaptions (boolean)
Language selector  → List of available VTT caption tracks
