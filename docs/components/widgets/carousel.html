Carousel / Slideshow Component

Multi-slide content widget supporting images, videos and text with auto-play, layout variants, and 3 visual styles for sequential content presentation.

Overview UML Class Sequence Component Flow Features Technical Data Model Files Settings Behaviors

Overview

The Carousel/Slideshow Component enables sequential content presentation with up to 12 slides. Each slide supports mixed media (images, videos, text) with configurable layout, auto-play, and 3 distinct visual styles (classic, pager, statusbar). The bxSlider library powers the animation and navigation engine, reinitializing dynamically when settings change.

Key Capabilities

  • 2–12 slides enforced (min 2, max 12)
  • 3 visual styles: Classic (arrows), Pager (dots), Status Bar
  • Auto-play with configurable pause duration 0–999 seconds
  • Mixed media per slide: image, video (local/YouTube), or text
  • Horizontal and vertical layout modes (auto-vertical on mobile <500px)
  • Multi-upload creates multiple slides up to the 12-slide maximum
  • Caption, title, header, and instruction visibility toggles

Behaviors & Constraints

  • Slide count enforced: addSlideClick() blocks at 12; deleteSlideClick() blocks at 2. Both enforce limits before array mutation.
  • bxSlider reinit: Changing visual style or adding/removing slides destroys and recreates the bxSlider instance via window.applystyleslideshow().
  • YouTube pause: On slide navigation, outgoing YouTube iframes receive postMessage({ event: 'command', func: 'pauseVideo' }) to stop playback.
  • Multi-upload batch: uploadMultipleSlideshow() creates one slide per uploaded file, stopping at the 12-slide maximum without error.
  • Mobile responsive: Horizontal layout automatically switches to vertical when the container width drops below 500px via CSS media query override.
  • Non-gradable: The slideshow is a content presentation widget; no SCORM scoring or student interaction state is tracked.
  • 3 style variants: Classic uses prev/next arrow buttons; Pager uses clickable dot indicators; Status Bar uses a linear progress bar strip.
