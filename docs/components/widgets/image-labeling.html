Component Overview

The Image Labeling component allows users to identify and label specific areas of an image using interactive hotspots, drag-and-drop labels, and coordinate-based positioning.

Flow Summary:

  • Editor Flow: Authors upload base images, add labels (2-20) with number or text types, position labels on specific areas, optionally draw arrow connectors with configurable styles, and select visual styles for the interface.
  • Student Flow: Students drag labels from a sidebar bank onto the image to correct positions, with arrow connectors rendering when configured, submitting when all labels are placed.
  • Validation: Label positions are validated against correct target zones with visual feedback (green highlights/checkmarks for correct placements, red highlights/crosses for incorrect), calculating overall score.
  • Data Flow: Label positions are tracked in an array and synchronized with SCORM, with Try Again returning only incorrect labels to the bank or Show Me displaying all correct positions automatically.
