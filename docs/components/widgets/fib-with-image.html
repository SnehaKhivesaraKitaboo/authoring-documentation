FIB with Image Component

Image-enhanced Fill in the Blank variant with a paragraph section and per-sentence images. Style locked to fibcard-style7. Shares the FIB codebase with extended media configuration.

Overview

FIB with Image is a variant of the Fill in the Blank component that adds a paragraph section and per-sentence image support. It shares the same directive (fibTemplate) and codebase as standard FIB but is identified by fibWithImage:true in settings. The visual style is permanently locked to fibcard-style7 which accommodates the image layout.

Key Capabilities

  • All standard FIB capabilities (dropdown/text/drag-and-drop)
  • Optional paragraph section with top/bottom media placement
  • Per-sentence image with left/right placement
  • Style locked to fibcard-style7 (image-compatible layout)
  • fibMediaLoc global variable tracks active image upload context
  • Detected via tocSelected=FIB with Image or fibStyle=fibcard-style7

Behaviors and Constraints

  • Style locked to style7: fibWithImage=true forces selected_style to fibcard-style7. The style thumbnail selector is hidden in the settings panel to prevent changes.
  • fibMediaLoc global variable: A global variable tracks whether the current image upload context is the "Paragraph" section or a "Sentence" section, routing uploads to the correct target.
  • Per-sentence limit: Each sentence supports a maximum of 1 image. Attempting to add a second image replaces the existing one rather than adding another.
  • Shared codebase detection: The directive detects FIB with Image mode via con.tocSelected=="FIB with Image" or con.fibStyle=="fibcard-style7" at initialization.
  • Paragraph section optional: The paragraph section (including its image) can be disabled — only the sentence-level content is required.
