Component Overview

Flow Summary:

  • Editor Flow: Authors choose pairing mode (1-to-1 or 1-to-n), create pairs (2-20) with text or image content in both columns, add multiple Column B items (1-3) per Column A item, link correct associations, and select from 4 visual styles.
  • Student Flow: Students connect items by dragging dots or clicking sequential pairs, creating multiple connections per Column A item in 1-to-n mode, with SVG lines visualizing all relationships.
  • Validation: Connections are validated with support for partial correctness in 1-to-n mode, displaying visual feedback (green/checkmarks for all correct, red/crosses for incorrect or partial), with both individual pair and generic feedback options.
  • Data Flow: Pair mappings object tracks all connections with line states, synchronized with SCORM, with Try Again clearing only incorrect lines or Show Me drawing all correct connections across complex 1-to-many relationships.
