Image-based hotspot activity where authors place interactive input zones over a background image. Students click hotspots to open modal editors and enter text or rich-text responses.

Overview

The Graphic Organizer Component lets authors place input hotspots over a background image at percentage-based coordinates. Students click hotspots to open a draggable modal editor where they enter text (plain or rich-text with Jodit). Authors can configure individual hotspot properties including size, color, lock state, and character limits. This is a non-scored component — answers are saved but not auto-graded.

Key Capabilities

  • Unlimited hotspots placeable over a background image
  • Percentage-based coordinates for responsive layout
  • Per-hotspot: size, background color (3 presets), outline color (3 presets), char limit
  • Lock for Student pre-fills and makes read-only
  • Advanced formatting via Jodit (ELA/Math toolbars)
  • Math via MathLive with Temml/LaTeX fallback
  • applyAll applies colors to all hotspots simultaneously

Behaviors and Constraints

  • Percentage coordinates: Hotspot positions (xcords/ycords) are stored as percentage strings (e.g., "25%") enabling responsive repositioning when the component container resizes.
  • Delete key conflict: Pressing the Delete key while a hotspot container is focused will delete the hotspot instead of characters in a text field — a known editing conflict.
  • Jodit ELA vs Math toolbars: When advancedFormatting=true, the Jodit toolbar configuration changes based on formattingValue: ELA shows text formatting tools; Math shows equation tools with MathLive integration.
  • applyAll colors all hotspots: The Apply for All button calls applyAll(value, index) which copies bgColor and outlineColor from the current hotspot to all other hotspots simultaneously.
  • No auto-grading: The component saves student answers via SCORM but does not compare them to inputAnswer for scoring — it is a content capture tool only.
  • Draggable modal: The student input modal is a draggable Bootstrap modal positioned over the image, allowing students to see the context while typing their answer.
