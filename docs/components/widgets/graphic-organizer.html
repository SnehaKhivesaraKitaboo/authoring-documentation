Component Overview

The Graphic Organizer component enables visual organization of information through an image-based interface with nodes, connections, and categories. It helps users structure concepts and relationships in a visual format.

Component Flow Diagram

This diagram illustrates the complete user interaction flow from authoring to student engagement:

Flow Summary:

  • Editor Flow: Authors upload a base image, add positioned hotspots (1-20) with resizable areas, create rich text content using Jodit editor, and optionally embed images or videos within hotspots.
  • Student Flow: Students explore the base image by clicking hotspot indicators to display content popups with rich text and media, navigating between hotspots to discover all information.
  • Validation: Non-assessed exploration widget tracks hotspot viewing completion, calculating percentage of hotspots accessed and marking activity complete when all have been viewed.
  • Data Flow: Viewed hotspot IDs are stored in an array and synchronized with SCORM, with completion percentage calculated dynamically for progress tracking and LMS reporting.
