Table GO Component

Structured data table widget with full row/column management, cell merging, Excel paste support, and optional student-editable cells with modal editors. Supports 4 visual themes and Jodit rich text for student input.

Overview

The Table GO Component provides a full-featured structured data table with up to 20 rows and 20 columns. Features include cell merging (rowspan/colspan), Excel paste via clipboard sanitization, alternating row colors, and student-editable cells that open a draggable modal editor with optional Jodit rich text. Authors use a right-click context menu to insert/delete/merge cells.

Key Capabilities

  • Up to 20 rows x 20 columns
  • Cell merge (rowspan/colspan) tracking in mergedCellsList{}
  • Excel paste sanitizes mso-* styles, preserves bold/italic/colors
  • Student-editable cells with draggable Bootstrap modal (2000 char limit)
  • 4 visual themes (style1-4-theme)
  • Right-click context menu: insert/delete rows/columns, merge/unmerge
  • Alternating row colors with 2 independent color pickers

Behaviors and Constraints

  • Creation dialog required: A dialog prompts for noOfRows and noOfCol before the grid renders. The check noOfRows==-1 detects first-time creation vs. existing data load.
  • Excel paste sanitization: pasteExcelGrid() strips all mso-* CSS properties (Microsoft Office inline styles) while preserving bold, italic, and color formatting. Auto-expands the table if pasted data exceeds current dimensions.
  • Cell merge tracking: mergedCellsList{} uses "row-col" string keys to store rowspan/colspan values and the original cell coordinates for unmerge operations.
  • Student-enabled cells: Cells marked as student-editable display a #113e9a blue border. Clicking opens a draggable Bootstrap modal with a 2000-character textarea and optional Jodit editor.
  • Selected cell highlight: Author-selected cells during editing display a #dedede background with a #714cda purple border for visual selection state.
  • Right-click context menu: A custom context menu replaces the browser context menu within the table editor, providing insert/delete row/column and merge/unmerge cell options.
