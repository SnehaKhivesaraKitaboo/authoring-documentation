Fill in the Blank (FIB) Component

Sentence-completion widget where students fill in blanks via dropdown, text input, or drag-and-drop. Supports 1-20 sentences with per-blank answer validation, optional media, and 7 visual styles.

Overview

The Fill in the Blank Component presents sentences with embedded blank spaces that students complete by selecting from a dropdown, typing text, or dragging and dropping answer chips. Blanks are embedded in sentence HTML using a special syntax. Supports up to 20 sentences and 20 blanks total, with 7 visual styles and optional sentence-level media.

Key Capabilities

  • 3 interaction types: Dropdown, Text input, Drag and Drop
  • 1-20 sentences, up to 20 blanks total
  • 7 visual style variants (fibcard-style1 through style7)
  • Per-blank correct answers stored Base64-encoded
  • Optional paragraph and per-sentence media (image)
  • Distractor answer options for dropdown/drag modes
  • Fisher-Yates shuffle for dropdown/drag choices

Behaviors and Constraints

  • fibradioChange rebuilds ALL blanks: Switching between Dropdown/Text/Drag modes calls fibradioChange() which destroys and regenerates all blank DOM elements — losing any unsaved student input.
  • Base64-encoded answers: Correct answers are stored as data-correct-answer attributes Base64-encoded (btoa) to prevent students from reading answers via browser devtools.
  • Critical bug - empty MathLive: If MathLive equation fields are present but empty, the package generation process throws a fatal error. Authors must either fill or remove all math fields before publishing.
  • FIB with Image locks style7: When fibWithImage=true, the selected_style is forced to fibcard-style7 and cannot be changed. Style selector is hidden in settings panel.
  • Dropdown flips on viewport: Dropdown menus detect viewport position and flip above or below the blank depending on available space.
  • Try Again partial credit: On non-final attempts, correct blanks lock in place; incorrect blanks clear allowing re-entry. Locked correct blanks cannot be changed.
