Multiple Choice Question (MCQ)

Standard multiple-choice question component supporting single and multiple correct answers with immediate feedback and validation.

Overview Architecture Use Cases Features Technical Details

Overview

The Multiple Choice Question (MCQ) component is a versatile assessment tool that presents learners with a question and multiple answer options. It supports both single-select (radio buttons) and multi-select (checkboxes) formats, making it suitable for various assessment scenarios.

Use Cases:

  • Knowledge checks and quizzes
  • Formative assessments
  • Practice exercises
  • Self-paced learning validation
  • Prerequisite testing

User Workflows:

  • Learner: Read question → Select option(s) → Submit answer → View feedback → Try again (if incorrect) → Review correct answer
  • Content Author: Configure question text → Add options → Set correct answers → Add media (images/video) → Apply styles → Set feedback messages
  • Instructor: View learner responses → Analyze performance metrics → Review correct/incorrect patterns

Flow Summary:

  • Editor Flow: Authors set question text with optional media (image, video, audio), add options (2-6) with text or images, mark correct answer(s) for single or multiple selection types, select from 6 visual styles and label types (A-B, 1-2, none), and configure per-option and generic feedback.
  • Student Flow: Students view questions with media, select answers using radio buttons (single) or checkboxes (multiple), with submit enabled when required selections are made.
  • Validation: Answers are validated with comprehensive visual feedback (green borders/checkmarks for correct, red borders/crosses for incorrect), supporting both individual option feedback and overall feedback messages.
  • Data Flow: Selected choices are tracked in an array and synchronized with SCORM, with Try Again clearing all selections and resetting visuals, or Show Me highlighting all correct answers and disabling interaction.

Key Features

  • Single & Multiple Answer Support
    Configure for single correct answer (radio) or multiple correct answers (checkbox)
  • Rich Media Support
    Include images, audio, and video in both questions and answer options
  • Immediate Feedback
    Provide instant correctness feedback with customizable messages and explanations
  • Randomization Options
    Randomize answer order to prevent pattern memorization
  • Retry Functionality
    Allow learners to retry questions with configurable attempt limits
