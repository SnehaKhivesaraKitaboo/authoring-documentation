Match The Pairs Component

Matching activity to connect related items from two columns.

Overview Architecture Use Cases Features

Overview

The Match The Pairs component presents two columns of items that users connect by drawing lines or clicking to create associations. It's ideal for vocabulary matching, concept pairing, and relationship identification activities.

User Workflows:

  • Learner: Views two columns of items, clicks or drags to create connections between related items, modifies connections if needed, submits for validation, and receives feedback on correct/incorrect matches.
  • Author: Creates pairs of related items, defines correct matches, controls whether items are shuffled, and configures visual layout and interaction style.

Flow Summary:

  • Editor Flow: Authors create pairs (2-10) with text or image content in Columns A and B, link correct associations, select from 4 visual styles, configure attempts and feedback, and enable optional Column B shuffling.
  • Student Flow: Students connect matching items between columns by dragging dots or clicking sequential items, with SVG lines drawn to visualize connections, submitting when all pairs are connected.
  • Validation: Connections are validated with visual feedback (green borders/checkmarks for correct, red borders/crosses for incorrect), supporting both individual pair feedback and generic overall feedback.
  • Data Flow: Pair connections are stored in an array and synchronized with SCORM, with Try Again clearing only incorrect connection lines or Show Me drawing all correct connections automatically.
